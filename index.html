プライバシーポリシー — リアルタイム文字起こし

最終更新日: 2026年2月18日

1. はじめに

「リアルタイム文字起こし」（以下「本拡張機能」）は、ブラウザタブの音声およびマイク音声をリアルタイムで文字起こしするChrome拡張機能です。本プライバシーポリシーは、本拡張機能がユーザーのデータをどのように取り扱うかを説明します。

2. 収集するデータ

2.1 OpenAI APIキー

本拡張機能は、文字起こし機能を提供するために、ユーザーが入力したOpenAI APIキーを収集します。APIキーはブラウザのローカルストレージ内にAES-GCM方式で暗号化して保存されます。APIキーはOpenAIのRealtime Transcription APIへのWebSocket接続認証にのみ使用されます。

2.2 音声データ

本拡張機能は、ユーザーが明示的に文字起こしを開始した場合に限り、以下の音声データを取得します。

  • ユーザーが共有を許可したブラウザタブの音声（PC音声）
  • ユーザーが有効化した場合のマイク音声

これらの音声データはOpenAIのRealtime Transcription APIにWebSocket（WSS）経由で直接送信されます。音声データは本拡張機能の開発者のサーバーには一切送信されません。音声データは拡張機能内に永続的に保存されません。

2.3 文字起こし結果

OpenAI APIから返された文字起こしテキストは、ブラウザのローカルストレージ（chrome.storage.local）に保存されます。このデータはユーザーの端末内にのみ存在し、外部には送信されません。

2.4 設定データ

言語設定、文字起こしモデル、区切り方、文字サイズ、マイク使用の有無などのユーザー設定は、ブラウザのローカルストレージに保存されます。

3. 収集しないデータ

本拡張機能は以下のデータを一切収集しません。

  • 個人を特定できる情報（氏名、メールアドレス、住所など）
  • ウェブ閲覧履歴、アクセスしたURL
  • ウェブページのコンテンツ
  • Cookie、位置情報、財務情報
  • 匿名の使用統計やアナリティクスデータ

4. データの使用目的

収集したデータは以下の目的にのみ使用されます。

  • OpenAI APIキー: OpenAI Realtime Transcription APIへの接続認証
  • 音声データ: OpenAI APIによるリアルタイム文字起こし処理
  • 文字起こし結果: ユーザーへの表示、コピー、エクスポート機能の提供
  • 設定データ: ユーザーの設定の保持・復元

5. データの共有・第三者への提供

音声データはOpenAI, Inc.のAPIに直接送信されます。OpenAIによるデータの取り扱いについては、OpenAIのプライバシーポリシーをご確認ください。

上記を除き、本拡張機能の開発者はユーザーのデータを第三者に共有、販売、転送することは一切ありません。

6. データの保存とセキュリティ

  • OpenAI APIキーはAES-GCM（256ビット）で暗号化してブラウザのローカルストレージに保存されます。
  • 音声データはWebSocket Secure（WSS）で暗号化して送信されます。
  • 文字起こし結果および設定データはブラウザのローカルストレージにのみ保存され、外部送信されません。
  • すべてのデータはユーザーの端末内にのみ存在します。

7. データの削除

ユーザーはいつでも以下の方法でデータを削除できます。

  • 拡張機能内の「クリア」ボタンで文字起こし履歴を削除
  • 設定画面でAPIキーを削除
  • 拡張機能をアンインストールすることで、すべてのデータが削除されます

8. Chrome Web Store Limited Use Policy への準拠

本拡張機能は、Chrome Web Store User Data Policyおよび Limited Use Requirements に準拠しています。取得したユーザーデータは、本拡張機能の単一の目的（リアルタイム文字起こし）を提供・改善するためにのみ使用されます。ユーザーデータを広告目的で使用すること、第三者に販売すること、信用力の判定に使用することはありません。

9. 子どものプライバシー

本拡張機能は13歳未満の子どもを対象としたものではなく、13歳未満の子どもから意図的に個人情報を収集することはありません。

10. 本ポリシーの変更

本プライバシーポリシーは必要に応じて更新されることがあります。重要な変更がある場合は、本ページの最終更新日を更新します。

11. お問い合わせ

本プライバシーポリシーに関するご質問は、Chrome Web Storeの拡張機能サポートページよりお問い合わせください。

Privacy Policy — Real-time Transcription

Last updated: February 18, 2026

1. Introduction

"Real-time Transcription" (hereinafter "this extension") is a Chrome extension that transcribes audio from browser tabs and microphone input in real time. This Privacy Policy explains how this extension handles user data.

2. Data We Collect

2.1 OpenAI API Key

This extension collects the OpenAI API key entered by the user to provide transcription functionality. The API key is encrypted using AES-GCM and stored in the browser's local storage. The API key is used solely for authenticating WebSocket connections to the OpenAI Realtime Transcription API.

2.2 Audio Data

This extension captures the following audio data only when the user explicitly initiates transcription:

  • Audio from the browser tab that the user has authorized for sharing (PC audio)
  • Microphone audio, if enabled by the user

Audio data is sent directly to the OpenAI Realtime Transcription API via WebSocket Secure (WSS). Audio data is never sent to the extension developer's servers. Audio data is not persistently stored within the extension.

2.3 Transcription Results

Transcription text returned from the OpenAI API is stored in the browser's local storage (chrome.storage.local). This data exists only on the user's device and is not transmitted externally.

2.4 Settings Data

User preferences such as language, transcription model, segmentation mode, font size, and microphone usage are stored in the browser's local storage.

3. Data We Do Not Collect

This extension does not collect any of the following:

  • Personally identifiable information (name, email address, physical address, etc.)
  • Web browsing history or visited URLs
  • Web page content
  • Cookies, location data, or financial information
  • Anonymous usage statistics or analytics data

4. How We Use Data

Collected data is used solely for the following purposes:

  • OpenAI API Key: Authentication for connecting to the OpenAI Realtime Transcription API
  • Audio Data: Real-time transcription processing via the OpenAI API
  • Transcription Results: Display, copy, and export functionality for users
  • Settings Data: Persisting and restoring user preferences

5. Data Sharing and Third-Party Disclosure

Audio data is sent directly to OpenAI, Inc.'s API. For information on how OpenAI handles data, please refer to OpenAI's Privacy Policy.

Other than the above, the extension developer does not share, sell, or transfer user data to any third parties.

6. Data Storage and Security

  • The OpenAI API key is encrypted with AES-GCM (256-bit) and stored in the browser's local storage.
  • Audio data is transmitted using WebSocket Secure (WSS) encryption.
  • Transcription results and settings are stored only in the browser's local storage and are not transmitted externally.
  • All data resides solely on the user's device.

7. Data Deletion

Users can delete their data at any time through the following methods:

  • Use the "Clear" button within the extension to delete transcription history
  • Remove the API key from the settings
  • Uninstall the extension to delete all data

8. Compliance with Chrome Web Store Limited Use Policy

This extension complies with the Chrome Web Store User Data Policy and Limited Use Requirements. User data obtained is used solely to provide and improve the single purpose of this extension (real-time transcription). User data is not used for advertising purposes, sold to third parties, or used for creditworthiness determinations.

9. Children's Privacy

This extension is not intended for children under the age of 13, and we do not knowingly collect personal information from children under 13.

10. Changes to This Policy

This Privacy Policy may be updated as necessary. If significant changes are made, the "Last updated" date at the top of this page will be revised.

11. Contact

For questions about this Privacy Policy, please use the support page on the Chrome Web Store extension listing.
